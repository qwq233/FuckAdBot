# Workspace Memory

- Go command environment: use `.gocache` for both `GOCACHE` and `GOTMPDIR`.
- Do not create a separate `.gotmp` or any other extra Go temp directory unless the user explicitly asks for it.
- Do not touch admin cache part.
- If you modify the database schema, ensure that you increment the database version and properly implement migration functions (unit tests are mandatory).
- Test & build artifacts should be placed in the out folder, such as CPU profiles generated by benchmarks or load test binaries.
